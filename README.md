# Calculator Application (HTML, CSS, JavaScript)

This is a simple and responsive calculator application built using HTML, CSS, and JavaScript.  
It performs basic arithmetic operations and demonstrates front-end fundamentals.


## Features

- Addition
- Subtraction
- Multiplication
- Division
- Clear display
- Responsive and modern UI


## Tech Stack

- HTML5
- CSS3
- JavaScript


## How to Run

1. Download or clone the repository
2. Open `index.html` in any modern web browser
3. Start using the calculator

*(No server or database required)*


## Project Highlights

- Clean and simple UI
- Lightweight and fast
- Beginner-friendly JavaScript logic
- No external libraries used
